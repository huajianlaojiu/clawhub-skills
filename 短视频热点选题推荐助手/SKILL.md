# 短视频热点选题推荐助手

## 技能概述
快速获取当日抖音、快手、B站热门话题，分析爆款规律，推荐适合你的短视频选题方向。

## 适用场景
- 创作者不知道今天拍什么
- 需要蹭热点但不知道怎么切入
- 想做垂直内容但缺乏选题灵感

## 使用方法
直接发送你的创作方向或关键词，例如：
- "推荐今天适合拍的短视频选题"
- "帮我找美食类热点话题"
- "最近什么话题容易火"

## 功能说明

### 1. 全平台热点获取
自动抓取抖音热榜、快手热榜、B站热门、微博热搜等多平台数据，筛选出当前最具热度的话题。

### 2. 选题匹配分析
根据你的账号定位（美食/穿搭/搞笑/知识/情感等），推荐最匹配的热点选题，并分析：
- 为什么这个话题会火
- 适合什么样的内容形式
- 如何切入才能脱颖而出

### 3. 爆款选题库
提供经过验证的爆款选题模板，直接套用：
- "对比类"选题公式
- "测评类"选题公式
- "教程类"选题公式
- "故事类"选题公式

### 4. 实时趋势预警
提醒即将爆发的潜力话题，抢占先机。

## 输出格式
```
📊 今日热点推荐

🔥 热搜话题TOP3
1. [话题名称] - 热度指数 🔥🔥🔥
2. [话题名称] - 热度指数 🔥🔥
3. [话题名称] - 热度指数 🔥🔥

💡 你的账号适合这些选题：
1. [选题方向] - 推荐理由
2. [选题方向] - 推荐理由

📝 爆款标题模板：
• "[话题]+反转/结局"
• "同事问我为什么..."
• "99%人都不知道的..."
```

## 示例对话
用户：推荐今天适合拍的美食类选题
助手：
```
🍜 今日美食热点选题推荐

🔥 当前热门话题：
• #淄博烧烤复刻 - 热度持续上升
• #自制神仙饮品 - 夏日清凉主题
• #打工人的早餐 - 共鸣感强

💡 推荐选题（高转化）：
1. "月薪3000vs月薪30000的早餐对比"
   - 理由：对立话题+视觉冲击，易引发讨论
   
2. "挑战30天不重样早餐 Day1-7"
   - 理由：连续剧式内容，培养用户追剧习惯
   
3. "同事吃了我的减肥餐惊呆了的反应"
   - 理由：反转+真实场景，易引发二次传播
```

## 注意事项
- 热点具有时效性，建议尽早行动
- 蹭热点要注意内容差异化，避免同质化
- 选择与账号定位相符的话题，避免粉丝流失
