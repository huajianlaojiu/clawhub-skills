# 视频封面设计提示词生成器

## 技能概述
根据视频主题和内容，智能生成适合各大平台的封面设计提示词（Prompt），用于AI绘图工具快速生成爆款封面图。

## 适用场景
- 创作者需要快速制作视频封面
- 不擅长设计但想有吸引力的封面
- 多平台分发需要不同尺寸封面

## 使用方法
发送视频主题或内容描述，例如：
- "帮我生成封面提示词：美食探店"
- "这个视频的封面怎么设计：[视频内容简介]"
- "生成适合抖音/快手/B站的封面提示词"

## 功能说明

### 1. AI绘图提示词生成
根据视频内容自动生成高质量的AI绘图提示词，包含：
- 画面构图描述
- 色彩风格建议
- 文字排版位置
- 光影氛围设定

### 2. 多平台尺寸适配
自动生成适合不同平台的封面尺寸提示词：
- 抖音/快手：9:16 竖版
- B站：16:9 横版
- 视频号：6:7 方版
- 小红书：3:4 方版

### 3. 爆款封面模板库
提供经过验证的爆款封面公式：
- "人物+大字报"模式
- "前后对比"模式
- "情绪表情"模式
- "悬念留白"模式

### 4. 热门风格推荐
根据当前流行趋势推荐设计风格：
- 高级感莫兰迪色系
- 赛博朋克霓虹风格
- 治愈系奶油风格
- 真实生活记录风格

## 输出格式
```
🎨 封面设计提示词

📐 推荐尺寸：9:16（抖音/快手）

🤖 AI绘图提示词：
[完整的英文提示词，包含画面描述、风格、光影等]

💡 设计要点：
• 突出人物表情或动作
• 主标题醒目，副标题补充
• 留白区域增加层次感

📝 可用中文字体风格：
• 综艺风描边字体
• 手写体圆字体
• 科技感金属字体

🎯 注意事项：
• 避免文字过多遮挡主体
• 颜色对比度要高，方便缩略图识别
```

## 示例对话
用户：帮我生成美食探店视频的封面提示词
助手：
```
🎨 美食探店封面设计提示词

📐 推荐尺寸：9:16（抖音/快手）

🤖 AI绘图提示词：
A beautiful food photography composition, close-up of delicious
steamed dumplings in a bamboo steamer, warm golden lighting,
cozy traditional Chinese restaurant background with soft bokeh,
steam rising from the food, overhead angle, vibrant colors,
cinematic food photography style, shallow depth of field,
high-end restaurant atmosphere, 9:16 vertical format

💡 设计要点：
• 食物占画面60%，蒸汽效果增加食欲感
• 右侧留白区域放标题文字
• 使用暖色调营造美食氛围

📝 中文标题建议：
• "探店｜藏在巷子里的神仙小馆"
• "被问了100次的宝藏餐厅🍜"
• "答应我一定要来这家！"

🎯 封面公式：
[美食特写] + [大字标题] + [emoji表情] = 高点击封面
```

## 提示词优化技巧
1. 越具体的描述，生成效果越好
2. 加入"high quality"、"4K"等词提升画质
3. 添加"photography"、"illustration"等词定义风格
4. 使用"8k, detailed"增加细节表现
